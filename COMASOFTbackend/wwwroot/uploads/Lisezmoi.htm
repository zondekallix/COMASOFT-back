Logiciel Adobe® Acrobat® XI Pro et Standard

 

Bienvenue dans le fichier Lisez-moi d’Adobe® Acrobat® XI Pro et d’Adobe Acrobat XI Standard, qui contient des informations essentielles sur les produits Adobe Acrobat. Dans ce fichier, les informations qui s’appliquent à tous les produits ne précisent aucun nom de produit. Pour obtenir des informations sur la prise en charge d’Adobe Acrobat XI, voir http://www.adobe.com/go/acrobat_support_fr

 

Configuration système

 

Configuration requise pour Acrobat XI Pro pour Windows® :

 

•         Processeur de 1,3 Gigahertz ou plus rapide

•         Microsoft® Windows XP avec SP3 pour 32 bits (avec SP2 pour 64 bits) ; Windows Server® 2003 R2 (32 bits et 64 bits) ; Windows Server 2008 ou 2008 R2 (32 bits et 64 bits) ; Windows 7 (32 bits et 64 bits) ; Windows 8 (32 bits et 64 bits)

•         512 Mo de mémoire vive installée (1 Go recommandé)

•         1,9 Go d’espace disponible sur le disque dur (Remarque : le programme d’installation crée des fichiers temporaires durant l’installation, de sorte que l’espace réellement requis sur le disque dur à ce moment-là est supérieur à la valeur indiquée.)

•         Résolution de l’écran : 1 024 x 768

•         Internet Explorer 7, 8, 9, 10 ; Firefox (ESR) ; Chrome

•         Lecteur de DVD-ROM, si installation à partir d’un DVD

•         Accélération matérielle vidéo en option

Pour Windows Server 2003 SP2 et Windows XP SP2 pour 64 bits, le correctif logiciel Microsoft KB-930627 est requis. Voir :

http://support.microsoft.com/kb/930627

 

 

Configuration requise pour Acrobat XI Standard pour Windows® :

 

•         Processeur de 1,3 Gigahertz ou plus rapide

•         Microsoft® Windows XP avec SP3 pour 32 bits (avec SP2 pour 64 bits) ; Windows Server® 2003 R2 (32 bits et 64 bits) ; Windows Server 2008 ou 2008 R2 (32 bits et 64 bits) ; Windows 7 (32 bits et 64 bits) ; Windows 8 (32 bits et 64 bits)

•         512 Mo de mémoire vive installée (1 Go recommandé)

•         1,5 Go d’espace disponible sur le disque dur (Remarque : le programme d’installation crée des fichiers temporaires durant l’installation, de sorte que l’espace réellement requis sur le disque dur à ce moment-là est supérieur à la valeur indiquée.)

•         Résolution de l’écran : 1 024 x 768

•         Internet Explorer 7, 8, 9, 10 ; Firefox (Extended Support Release) ; Chrome

•         Lecteur de DVD-ROM, si installation à partir d’un DVD

•         Accélération matérielle vidéo en option

Pour Windows Server 2003 SP2 et Windows XP SP2 pour 64 bits, le correctif logiciel Microsoft KB-930627 est requis. Voir :

http://support.microsoft.com/kb/930627

 

Instructions d’installation

 

Suivez les instructions ci-dessous pour installer le nouveau logiciel ou pour effectuer la mise à niveau à partir d’une version antérieure. Si vous installez une mise à niveau, vous aurez peut-être à fournir une preuve de propriété.

 

Installation d’Acrobat XI sous Windows

 

1. Redémarrez Windows et désactivez le logiciel antivirus, le cas échéant.

 

2. Connectez l’ordinateur à Internet.

 

3. Démarrez l’installation :

  • Si vous avez acheté Acrobat sur un DVD, insérez le DVD.
  • Si vous avez acheté Acrobat sous forme de téléchargement, téléchargez le package sur le disque dur.

 

Adobe vous recommande de sauvegarder le package de téléchargement, de préférence sur un DVD, avant d’installer le logiciel. Lorsque vous double-cliquez sur le fichier .exe pour le décompresser :

 

Sous Windows XP,

 

le dossier <lecteur système>:\Documents and Settings\<utilisateur>\Bureau\Adobe Acrobat XI est créé sur le Bureau.

 

Sous Windows 7 et 8,

 

le dossier <lecteur système>:\utilisateurs\<nom d’utilisateur>\Bureau\Adobe Acrobat XI est créé sur le Bureau.

 

Pour commencer l’installation, ouvrez le dossier Adobe Acrobat XI et double-cliquez sur setup.exe.

 

Dans la boîte de dialogue de configuration d’Adobe Acrobat XI, sélectionnez une langue, puis cliquez sur OK. Veillez à sélectionner la langue que vous avez achetée. Sur l’écran de configuration d’Adobe Acrobat XI, cliquez sur Suivant. Le programme d’installation calcule l’espace disque requis et vérifie la présence de versions antérieures. Si une version antérieure du même produit Acrobat est détectée, cliquez sur Continuer pour la désinstaller.

 

4. Sur l’écran de configuration, entrez votre nom, votre société et le numéro de série du produit. Vous n’avez pas à entrer un numéro de série si vous utilisez le produit en mode d’évaluation ou si vous avez acheté un abonnement (Pro uniquement). Si vous entrez un numéro de série, veillez à ce que l’ordinateur soit connecté à Internet pour la validation en ligne. Remarque : l’installation en mode d’évaluation vous permet d’utiliser le logiciel pendant la période d’évaluation de 30 jours à compter du premier lancement.

 

5. Entrez le numéro de série de votre copie du programme. Le numéro de série pour les achats de licence individuelle se trouve à l’extérieur du boîtier du DVD ou sur le reçu pour les achats avec livraison électronique réalisés sur l’Adobe Store. Vous trouverez également le numéro de série des achats de licence individuelle avec livraison électronique dans votre historique de commandes en ligne, à l’adresse suivante : http://www.adobe.com. Le numéro de série pour les achats effectués sur le site Web de gestion des licences Adobe se trouve dans votre dossier, à l’adresse suivante : http://licensing.adobe.com.

 

6. Sur l’écran d’activation de la licence Adobe, cliquez sur Suivant. L’activation est une série anonyme d’étapes simples et rapides exécutées par le logiciel. Pour en savoir plus sur l’activation du produit, rendez-vous sur le site Web d’Adobe à l’adresse suivante : http://www.adobe.com/go/activation_fr/

 

7. Sélectionnez un type d’installation :

  • Par défaut (type sélectionné par défaut) pour installer les fichiers du programme pour Acrobat, Acrobat Distiller, PDFMaker, le pilote d’imprimante Adobe PostScript correspondant à votre système, les modules externes standard et les accessoires.
  • Complète pour installer toutes les fonctionnalités d’Acrobat XI (l’espace disque requis est supérieur).
  • Personnalisée pour indiquer quels composants installer.

 

Remarque : pour obtenir une description d’un composant, sélectionnez Personnalisée, puis cliquez sur Suivant. Sélectionnez ensuite un composant pour afficher sa description. Cliquez sur Précédent pour revenir à l’écran de sélection du type d’installation si vous ne souhaitez pas procéder à une installation personnalisée.

 

Cliquez sur Suivant. Modifiez le répertoire d’installation (le cas échéant) ou conservez le répertoire par défaut affiché, puis cliquez sur Suivant.

 

8. Cliquez sur Installer pour lancer l’installation. Si des processus en cours d’exécution sont en conflit avec l’installation, vous êtes invité à les interrompre avant de poursuivre l’installation. Fermez le processus, puis cliquez sur le programme d’installation pour poursuivre.

 

9. La dernière boîte de dialogue, Installation terminée, s’affiche. Cliquez sur Terminer. Vous pouvez maintenant démarrer Acrobat.

 

10. Au démarrage, acceptez le contrat de licence de l’utilisateur final afin de pouvoir utiliser le logiciel.

 

11. Si vous avez indiqué un numéro de série durant l’installation, reportez-vous à l’étape 12 ci-dessous. Si vous utilisez Acrobat en mode d’évaluation ou par abonnement (Pro uniquement), l’écran Sign In Required (Connexion requise) s’affiche. Sur cet écran, cliquez sur Sign in now (Se connecter maintenant). Sur l’écran qui s’affiche, entrez vos informations d’identification Adobe. Si vous n’avez pas d’ID Adobe, cliquez sur Créer un ID Adobe et entrez les détails demandés. Cliquez ensuite sur Créer. Suivez les instructions à l’écran pour terminer le processus de création. Cliquez sur Sign In (Se connecter). Vous devriez maintenant pouvoir utiliser Acrobat.

 

12. Si vous avez entré un numéro de série durant l’installation, l’écran Register (Enregistrer) s’affiche. Sur cet écran, entrez vos informations d’identification Adobe. Si vous n’avez pas d’ID Adobe, cliquez sur Créer un ID Adobe et entrez les détails demandés. Cliquez ensuite sur Créer. Suivez les instructions à l’écran pour terminer le processus de création d’ID Adobe et le processus d’enregistrement. Vous pouvez également appuyer sur Skip (Ignorer). IMPORTANT : vous devez être connecté à Internet pour activer votre copie du logiciel. Vous devez activer votre copie sous 30 jours. Passé ce délai, le logiciel ne fonctionnera plus.

 

Désinstallation d’Acrobat XI sous Windows

 

1. Avant de désinstaller Acrobat XI, pensez à désactiver le produit. Pour ce faire, sélectionnez Aide > Désactiver. Ensuite, pour désinstaller le logiciel, ouvrez le Panneau de configuration et sélectionnez Ajout/Suppression de programmes.

 

2. Dans la liste des applications qui s’affiche, sélectionnez Acrobat XI et cliquez sur le bouton Supprimer/Désinstaller.

 

3. Dans la boîte de dialogue qui s’affiche, confirmez la suppression d’Acrobat. Sélectionnez Oui pour désinstaller le produit.

Enregistrement
